# XIM Embeds

## Price Widget

To use the XIM Price widget past the following code into your page:

```html
<iframe src="//pselden.github.io/xim-embeds/price/" height="190" width="160" frameborder="0" allowtransparency="true" scrolling="no"></iframe>
```
